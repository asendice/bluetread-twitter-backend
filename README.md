# bluetread-twitter-backend

## Installation
```
# Clone this repository
$ git clone https://github.com/asendice/bluetread-twitter-backend.git

# Go into repository
$ cd bluetread-twitter-backend

# Install backend dependecies
$ npm install

# Run server
$ npm start

```
